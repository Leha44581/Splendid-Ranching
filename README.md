### Contributing to an existing repository that uses this template
1. Create a prism launcher instance for desired version and loader
2. Open the minecraft folder inside of the instance folder `(instancename)/minecraft/` with a terminal and run `git clone https://github.com/Deepacat/jeopackjam1.git`
3. Copy `(instancename)\minecraft\.pakku\prism-overrides` into your `(instancename)` folder
4. You now have a functioning repo/game instance to modify and commit from
